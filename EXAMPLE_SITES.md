# Better Sites for Language Examples

## Current: Tatoeba
**Pros:**
- Free and open source
- Community-verified translations
- Audio pronunciations
- Many language pairs

**Cons:**
- Limited examples for some languages
- Quality varies
- No frequency data

## Better Alternatives:

### 1. **Linguee** (https://www.linguee.com)
- **Best for:** European languages (German, French, Spanish, etc.)
- Shows real-world examples from bilingual texts
- Frequency information
- Context-rich examples
- **Limitation:** Limited language coverage

### 2. **Reverso Context** (https://context.reverso.net)
- **Best for:** European + Arabic, Japanese, Chinese
- Examples from movies, books, news
- Translations with context
- **Limitation:** Ads, limited languages

### 3. **OPUS** (http://opus.nlpl.eu)
- **Best for:** Academic/research use
- Massive parallel corpora
- Multiple languages
- **Limitation:** Not user-friendly, requires technical knowledge

### 4. **Sketch Engine** (https://www.sketchengine.eu)
- **Best for:** Linguistic research
- Concordance examples
- Frequency data
- **Limitation:** Paid service, complex

### 5. **WordReference** (https://www.wordreference.com)
- **Best for:** Spanish, French, Italian, Portuguese, German, Chinese
- Forum discussions
- Real examples
- **Limitation:** Limited languages

### 6. **YouGlish** (https://youglish.com)
- **Best for:** All languages (uses YouTube)
- Real pronunciation examples
- Context videos
- **Limitation:** Audio only, not text examples

## Recommendation:
Keep **Tatoeba** as default (best balance of languages/examples), but add **Linguee** for European languages where available, and **WordReference** for major languages.
